# revel-mvp
Prototipo PWA gamificada para entrenamiento de inteligencia emocional.
Objetivo: Validar la experiencia narrativa y la retención.

Estructura inicial:
- /src : código frontend (React/Next)
- /public : assets
- /scripts : scripts de despliegue y build
- /docs : guiones, prompts y manual de QA

Próximo paso: implementar la primera historia interactiva (Capítulo 1).
